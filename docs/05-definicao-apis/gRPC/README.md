# gRPC APIs

Definicoes de servicos gRPC usando Protocol Buffers.

## Convencoes

- `{service}.proto`: Arquivo protobuf
- `{service}.proto.md`: Documentacao do servico

## Estrutura

Cada servico deve incluir:

1. Definicao do package
2. Imports necessarios
3. Messages (request/response)
4. Service definition
5. Comentarios explicativos
