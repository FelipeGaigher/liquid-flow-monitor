# REST APIs

Especificacoes de APIs REST do sistema.

## Convencoes

- `{service}-api.md`: Documentacao em Markdown
- `{service}-openapi.yaml`: Especificacao OpenAPI 3.0

## Estrutura de Documentacao

Cada API deve documentar:

1. Base URL
2. Autenticacao
3. Endpoints (metodo, path, descricao)
4. Request/Response schemas
5. Codigos de erro
6. Exemplos
